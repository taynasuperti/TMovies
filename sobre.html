Qual é o objetivo do projeto e como ele foi feito?

Este projeto foi desenvolvido com o objetivo de criar uma plataforma informativa para a apresentação de filmes e séries, utilizando HTML, CSS e JavaScript. A principal proposta é proporcionar aos usuários uma experiência simples e eficiente ao procurar e visualizar informações sobre filmes e séries de forma dinâmica e atraente.

O projeto busca integrar dados de filmes e séries retirados de uma API externa, sendo ela a The Movie Database API para apresentar informações detalhadas sobre cada item, incluindo:

Título;

Descrição;

Gênero;

Imagem de capa;

Data de lançamento;

Trailers.
